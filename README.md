## Getting Started

Follow these instructions to set up and run the portfolio website locally on your machine.

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/Harmon1713/portfolio.git
   cd portfolio
   ```

2. **Open the project folder** in your preferred code editor.

### Running Locally

#### Option 1: Open with Web Browser

1. **Locate the index.html file**:
   - Navigate to the `index.html` file in your project directory.

2. **Open the file in your browser**:
   - Right-click on the `index.html` file.
   - Select "Open with" and choose your preferred web browser.

#### Option 2: Use a Simple HTTP Server

   ```bash
   python -m http.server
   ```

 - Open your browser and go to `http://localhost:8000`.

## Usage

### Language Toggle

- **Switch languages** by clicking on the language buttons (English, Spanish, Portuguese) in the header.
- The website remembers your preferred language using local storage.

### Search Functionality

- **Search by skills** using the search bar.
- Enter skills separated by commas to filter projects based on your input.
- To view all projects, type "all," "todo," or "todos" based on the selected language.

### Interactive Projects

- **View project details** by clicking on the icons or project categories.
- Each project includes a description, technologies used, and a link to the source code or demo.

### Draggable Travel Map

- **Explore the travel map** by dragging within the interactive map section.
- Zoom in/out for a closer look at visited locations.
