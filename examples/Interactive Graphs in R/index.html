My Interactive Graphs

Libraries Used:

  • ggplot2: A plotting system for R, based on the grammar of graphics, which makes it easy to create complex multi-layered graphics.
  • ggiraph: An extension of ggplot2 that allows for the creation of interactive graphics.
  • patchwork: A library that makes it easy to combine multiple ggplot2 plots into a single cohesive visualization.
  • dplyr: A data manipulation package that provides a consistent set of verbs to help solve the most common data manipulation challenges.
  • htmlwidgets: A package that provides a framework for creating HTML widgets that can be used in R.

Interactive ggiraph Graph with Customized Tooltip

Dataset Used

  • mpg: A dataset that comes with ggplot2, which contains fuel economy data for various car models.

Code Breakdown

  • mpg dataset: Used to create a scatter plot where displ (engine displacement) is plotted against hwy (highway miles per gallon). The points are colored by the class of the car.
  • Customized tooltip: Enhanced dataset by adding a new tooltip column that concatenates the manufacturer, model, and class of each car. The new tooltip column is used to provide more detailed information in the tooltip.

Linked Interactive Graphs

Dataset Used

  • mtcars: A built-in dataset in R that contains various automobile data extracted from the 1974 Motor Trend US magazine.

Code Breakdown

  • mtcars dataset:
    • p1: A scatter plot where wt (weight) is plotted against mpg (miles per gallon), with tooltips and data IDs set to the row names of the dataset.
    • p2: Another scatter plot where drat (rear axle ratio) is plotted against qsec (1/4 mile time), with similar tooltip and data ID settings.
    • p3: A bar plot of cyl (number of cylinders), with data IDs set to the row names.
  • Linked Plots: The three plots are combined into a single interactive visualization using patchwork.
