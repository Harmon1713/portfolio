Understanding Height Dynamics Through Regressions


R (ggplot2, lm(), gvlma, predictmeans, car, caret), Python (Pandas, NumPy, SciPy, Matplotlib, Seaborn, statsmodels, pylab)


The project "Understanding Height Dynamics Through Regressions" examines the effect of gravitational compression on human height by analyzing morning and evening measurements from students at a boarding school in India, aiming to quantify this impact using regression analysis. Technical skills used included data wrangling with Pandas and dplyr, numerical operations with NumPy, statistical testing with SciPy, and data visualization with Matplotlib, Seaborn, and ggplot2. Linear regression models were fitted using Statsmodels in Python and the lm function in R. Assumptions testing included checking for linearity, normality (Shapiro-Wilk test), and homoscedasticity (Breusch-Pagan test), as well as identifying influential data points using Cook's distance and leverage values. The analysis confirmed a strong positive linear relationship between AM_Height and PM_Height, with models explaining a significant portion of the variance in evening heights. This project demonstrated proficiency in data analysis, statistical modeling, and assumption validation using both R and Python, highlighting the measurable impact of gravitational compression on daily height variations.

Predicting Gold Presence Through Antimony Levels Using Logistic Regression


R (caret, magrittr, dplyr, tidyr, ggplot2, lmtest, popbio, e1071, readxl)


In this project, logistic regression was employed to predict the presence of gold in a given location based on antimony levels. The analysis involved the use of several R packages, including caret for creating the confusion matrix, dplyr and tidyr for data manipulation, and ggplot2 for visualization. The model was built using the glm function, specifying a binomial family to handle the binary nature of the response variable (presence of gold). After fitting the model, predictions were made, and a confusion matrix was generated to assess the model's accuracy. Additionally, diagnostic checks were performed to evaluate model assumptions, including the Durbin-Watson test for autocorrelation and the inspection of residuals and influence measures. Visualizations were created to assess the relationship between the log-odds (logit) and predictor variables, aiding in the interpretation of the logistic regression model. The results indicated that higher antimony levels are associated with an increased likelihood of finding gold, as shown by the positive coefficient for antimony in the logistic regression model. The model's performance, as evaluated by the confusion matrix, suggested it was reasonably accurate in predicting gold presence, making it a useful tool for geologists in identifying potential gold deposits.
